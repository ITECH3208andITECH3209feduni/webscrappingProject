Configure setups:
1. You have a requirements.txt file install all the lib included there by using the command given below:
    pip install -r requirements.txt
    (Note: Python must be installed and set as environment variable at first)
2. Install Django:-
    inside webscrapping (cd webscrapping)
    pip install django
3. python manage.py migrate


Run the project in localhost i.e 127.0.0.1:8000/
python manage.py runserver
